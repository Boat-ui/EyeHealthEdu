# Eye Health Edu

**Educational Eye Health Screening App**

⚠️ **IMPORTANT DISCLAIMER:** This app is for **EDUCATIONAL PURPOSES ONLY**. It demonstrates basic pattern recognition concepts and is **NOT** a medical diagnostic tool.

## Features
- Camera/image upload functionality
- Educational AI pattern analysis simulation
- Eye health information
- Doctor locator (external links)

## Educational Purpose
This app demonstrates:
- How image processing works
- Basic color pattern analysis
- The difference between educational simulation vs real medical AI
- Importance of professional medical consultation

## Technology
- Android Studio
- Java
- Basic image processing algorithms

## Legal & Ethical Notes
- This app **cannot** diagnose any medical condition
- Always consult qualified healthcare professionals
- For educational demonstration only
- Not FDA approved or medically validated

## Installation
1. Clone repository
2. Open in Android Studio
3. Build and run on Android device/emulator

## Warning
DO NOT use this app for medical decisions. It is a technology demonstration, not a healthcare tool.
